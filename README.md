|        |      |      |       |      |     |        |            |
|        |      |      |       |      |     |        |            |
|        |      |      |       |      |     |        |            |
|        |      |      |       |      |     |        |            |
|        |      |      |       |      |     |        |            |

For the purposes of our first checkpoint, we will start with 100 days of historical data.
